# 📝 Changelog - Kitten Tennis

## v2.1.0 (Текущая версия) ✨

### 🐛 Исправлено

**Физика мяча:**
- ✅ Размер мяча увеличен с 22 до 28 пикселей (теперь виден лучше)
- ✅ Улучшена коллизия мяча с игроками (добавлена проверка деления на 0)
- ✅ Мяч больше не проходит сквозь игроков
- ✅ Улучшены отскоки от земли и стен
- ✅ Коэффициент отскока изменён с 0.80 на 0.78 (более плавно)

**Управление:**
- ✅ Поддержка русской раскладки:
  - `Ф` вместо `A` (влево)
  - `В` вместо `D` (вправо)
  - `Ы` вместо `S` (вниз)
- ✅ Одновременно работают обе раскладки (англ + русс)
- ✅ Добавлена обработка всех вариантов управления

**Игромеханика:**
- ✅ Прыжок уменьшен с -580 до -450 (реалистичнее)
- ✅ Улучшена синхронизация состояния игроков
- ✅ Глаза на персонажах теперь правильно отображаются

### 🎨 Новые фичи

**Система скинов:**

- ✨ Скины сохраняются при смене режима (локально в памяти)

**Режим с ботом:**
- 🤖 Новая кнопка "Играть с ботом" в главном меню
- 🤖 Простой но эффективный AI:

  - Занимает оптимальную позицию для удара
  - Прыгает в нужный момент
  - Реагирует на быстрый мяч
- 🤖 Идеально для одиночной тренировки
- 🤖 Регулируется сложность (можешь изменить скорость в коде)

## 🎯 Как это использовать

1. **Локально**: `npm start` → `localhost:3000`
2. **На Render**: Загрузи на GitHub → Deploy на Render
3. **С друзьями**: Поделись кодом комнаты

## ⚠️ Известные ограничения

- Мяч может глитчить при очень низких fps
- Бот не учится (всегда одинаковый уровень)
- Нет сохранения данных между сеансами
- Максимум 50 комнат одновременно (зависит от памяти сервера)
